# Khus Sweets

A modern, high-performance website built for **Khus Sweets** to showcase premium Indian sweets, cakes, bakery items, snacks, and restaurant offerings with a strong focus on branding, user experience, and SEO.

## Overview

Khus Sweets is a digital-first sweets shop website designed to help customers explore products, discover offers, view menus, and connect with the business easily across devices.

This project combines elegant UI design with modern frontend engineering practices to create a fast, responsive, and conversion-focused web presence.

## Features

- Modern and responsive user interface
- Mobile-first design approach
- Premium sweets & bakery brand presentation
- Product and menu showcase sections
- Restaurant / food service sections
- SEO-optimized architecture
- Sitemap, robots, metadata, and structured schema support
- Fast page loads and optimized assets
- Smooth animations and polished interactions
- Contact and business discovery sections

## Tech Stack

- Next.js
- React.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel Deployment

## SEO & Performance

Built with strong technical SEO foundations:

- Metadata optimization
- Open Graph & social sharing
- Sitemap.xml
- Robots.txt
- Structured Data (JSON-LD)
- Canonical URLs
- Performance optimization
- Mobile responsiveness
